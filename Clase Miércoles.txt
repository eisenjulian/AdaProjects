JS no es un lenguaje especificado, no necesito declarar el tipo de dato, puedo poner cualquiera de ellos. 
Tipos de datos JS: valor que puede tener una variable.
   	var nombre = valor;
El valor puede ser: 
- string "" ó ''
- numbers
- boolean: true/false
- array: ["" , 2 , true]
- function:
- objects:
- null: Asigno este valor para especificar que la variable está declarada.
- undefined: La variable puede estar sin valor o no declarada, no es claro. Ambiguo.

OBJETOS
	var humano = {
		cantOjos : 2;
		nombre : "Estefanía",
		respiran : function() {
			console.log("");
			},
		alimentación : ["vegetariano", "carnívoro"],
		extraterrestre : false

	}

Para llamar el valor Estefanía:
	humano.nommbre
Cambiar el valor de un atributo:
	humano.cantOjos = 1;

Las funciones dentro de objetos se llaman MÉTODOS.
